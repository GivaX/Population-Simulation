To compile this program you need:

main.cpp

Cell.h
Residential.h
Commercial.h
Industrial.h
Pollution.h
Analysis.h

Cell.cpp
Residential.cpp
Commercial.cpp
Industrial.cpp
Pollution.cpp
Analysis.cpp

config1.txt
region1.csv

Import these files to the CELL machines, then run this command:

g++ -o Project1 *.cpp

Then run this command:
./Project1
